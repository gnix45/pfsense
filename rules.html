Detailed Guide: Creating a Firewall Rule

Step 1: Access the Rules Page

From your pfSense dashboard, navigate to **Firewall** and then select **Rules**. This is where you'll see a list of all your existing rules, organized by interface (WAN, LAN, etc.).

Step 2: Add a New Rule

Select the tab for the interface you want the rule to apply to (e.g., LAN). Click the **"Add"** button at the top of the rule list.

Step 3: Configure the Rule's Action

The most important setting is the **Action**.

  • **Pass:** Allows the traffic to pass.
  • **Block:** Silently drops the traffic. The sender receives no notification.
  • **Reject:** Drops the traffic and sends a "port unreachable" message back to the sender.

Step 4: Define Source and Destination

This is where you specify **what** and **where** the rule applies.

  • **Interface:** Set to `LAN` or `WAN`.
  • **Address Family:** `IPv4` or `IPv6`.
  • **Protocol:** `TCP`, `UDP`, or `any`.
  • **Source:** You can select a single host, a network (`LAN net`), or an alias.
  • **Destination:** Similar to the source, this can be a single IP, a network, or a port.

Step 5: Save and Apply

Once you've configured your rule, click **Save**. You must then click the **Apply Changes** button at the top of the rules page for the new rule to become active.
